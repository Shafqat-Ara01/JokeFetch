JokeFetch is a fun and interactive mini project that fetches random jokes from the **icanhazdadjoke API**.  
It includes a smooth **loading spinner**, **copy-to-clipboard**, and a fully functional **light/dark mode** with `localStorage`.

Perfect for practicing API calls, DOM manipulation, and UI/UX polish.

---

##  Features

-  **Fetch Random Jokes** using a public API  
-  **Loading Spinner** while fetching  
-  **Copy Joke** to clipboard with feedback message  
-  **Light/Dark Mode** (saved using localStorage)  
-  Clean and responsive UI  
-  Fast and beginner-friendly code structure

- **HTML5**
- **CSS3** (custom UI + animations)

- **icanhazdadjoke API**
